Setup Guide

Follow our step-by-step instructions to configure your printer with Scanner2Email.

HP

HP Printer Setup

1

Access Printer Settings

On your HP printer's touchscreen:

  • • Touch the Setup icon (wrench or gear)
  • • Select Network Setup
  • • Choose Email Server or Scan to Email
2

STARTTLS
3

Enter Authentication

Use your Scanner2Email credentials from the dashboard:

Generate your credentials from the Scanner2Email Dashboard under SMTP Settings.

4

Configure Recipient Email

Set the default "To" address on your printer:

  • • Enter the email address you want scanned documents sent to
  • • This address must be in your Scanner2Email recipient whitelist
  • • Add recipients via the Dashboard before scanning

Tip: Most HP printers have a "Test Email" option. Use it to verify your settings before scanning important documents.

BR

Brother Printer Setup

1

Open Web Based Management

Access your Brother printer's embedded web server:

  • • Find your printer's IP address from the settings menu
  • • Enter the IP address in a web browser
  • • Log in (default credentials are often admin / access)
2

Navigate to Scan to Email

In the web interface:

  • • Go to Scan → Scan to Email → SMTP Settings
  • • Or Network → Protocols → SMTP
3

Enter SMTP Configuration

smtp.scanner2email.com
587
STARTTLS
4

Configure Sender & Recipient

Fill in the authentication and recipient fields:

  • • From Address: Use your Scanner2Email SMTP username
  • • User ID / Auth: Same SMTP username
  • • Password: Your generated SMTP password
  • • To Address: An email on your recipient whitelist

Tip: Use STARTTLS, not SSL/TLS. Brother's menu can be confusing — look for SMTP Port 587 with TLS enabled, not SSL port 465.

Open Printer Settings

Set Send From to your SMTP username and Send To to your whitelisted recipient.

4

Test the Connection

  • • Epson printers offer a Test Email option
  • • Run the test to verify the connection
  • • If it fails, check your credentials and network connectivity

Tip: Keep SSL/TLS set to OFF when using port 587. Epson handles encryption automatically via STARTTLS.

Canon Printer Setup

1

Access the Web Interface

Canon printers are best configured via their web interface:

  • • Press the Settings button on your printer
  • • Navigate to Network Settings → IP Address
  • • Enter the IP address in a web browser from any computer on your network
2

Navigate to Email Settings

In the web interface:

  • • Select Network Settings
