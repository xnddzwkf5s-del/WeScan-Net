1. Service Description

WeScan provides an SMTP relay service that allows authorised multifunction printers and scanners to send scanned documents as email attachments to a whitelist of approved recipients. The Service consists of:

  • An SMTP relay server at smtp.wescan.net (port 587 with STARTTLS)
  • A web dashboard for managing recipients and credentials
  • Account authentication via email OTP or OAuth providers
